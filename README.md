# GoogleKeep

## Project setup
```
yarn install or yarn
```

### Compiles and hot-reloads for development
```
yarn run serve
yarn run start
```

### Compiles and minifies for production
```
yarn run build
```

### Package for windows
```
yarn run package-win
```

### Lints and fixes files
```
yarn run lint
```

### Run your unit tests
```
yarn run test:unit
```

### Run your end-to-end tests
```
yarn run test:e2e
```
